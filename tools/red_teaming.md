# Red Teaming Tools

This section contains tools tailored for automated or dynamic red teaming to uncover vulnerabilities in Large Language Models.

- [PyRIT](https://github.com/Azure/PyRIT) - The Python Risk Identification Tool for generative AI developed by Microsoft to automate multi-turn red teaming.
- [promptfoo](https://github.com/promptfoo/promptfoo) - A command line and library for testing and evaluating LLM application security via customizable attack variants.
- [Garak](https://github.com/leondz/garak) - A vulnerability scanner for LLMs (Language Model Assessment) covering various adversarial and failure modes.
