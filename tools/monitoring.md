# Monitoring & Observability

Tools to monitor LLM operations, observe interactions post-deployment, and gain insights into operational anomalies.

- [Langfuse](https://langfuse.com/) - Open source LLM observability and evaluation framework.
- [Trulens](https://github.com/truera/trulens) - Evaluation and tracking for LLM applications.
- [Opik](https://github.com/comet-ml/opik) - An open platform to log, trace and observe LLM interactions.
