# Agentic Security

Dedicated tools to assess the safety and security posture of autonomous agents and integrations relying on the Model Context Protocol (MCP).

- [Agentic Radar](https://github.com/mttaggart/agentic-radar) - A scanning tool specifically oriented toward uncovering vulnerabilities within LLM tool use.
- [Invariant](https://github.com/invariantlabs-ai/invariant) - A security analyzer for AI Agents, scanning for prompt injections, data leaks, and insecure tool accesses.
- [MCP-Shield](https://github.com/Puliczek/awesome-mcp-security) - Represents concepts and controls related to enforcing security on Model Context Protocol interfaces.
