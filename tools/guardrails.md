# Guardrails & Filters

Tools designed to provide safety boundaries, access control, and preventative measures before LLM interactions are executed.

- [NeMo Guardrails](https://github.com/NVIDIA/NeMo-Guardrails) - An open-source toolkit by NVIDIA for adding programmatic safeguards to LLM conversational systems.
- [Llama Guard](https://ai.meta.com/research/publications/llama-guard-llm-based-input-output-safeguard-for-human-ai-conversations/) - An LLM-based input/output safeguard from Meta.
- [Azure AI Prompt Shields](https://learn.microsoft.com/en-us/azure/ai-services/content-safety/concepts/jailbreak-detection) - Mechanisms to protect AI systems from jailbreaks and prompt injection attacks on the Azure platform.
