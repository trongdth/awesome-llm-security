# Benchmarks & Evaluation

This repository section catalogs standard datasets and frameworks to evaluate the safety, security, and alignment of Large Language Models and AI Agents.

## Current Benchmarks

- [JailbreakBench](https://jailbreakbench.github.io/) - A unified leaderboard tracking the jailbreak resistance of commercial and open-source models.
- [ZeroDayBench](https://arxiv.org/abs/2603.02297v1) - Evaluates an AI Agent's capability to discover, exploit, and patch zero-day vulnerabilities in real-world source code.
- [MSB (MCP Security Bench)](https://openreview.net/forum?id=irxxkFMrry) - The first benchmark designed to evaluate the security of the Model Context Protocol (MCP).
- [Existential Threats Benchmark](https://arxiv.org/html/2601.19970v1) - A benchmark assessing potential existential threats posed by powerful AI models.
- [AgentLeak](https://huggingface.co/papers/2602.11510) - Evaluating privacy leakage in multi-agent systems.

To understand how to contribute your own dataset, please refer to the main repository [CONTRIBUTING.md](../CONTRIBUTING.md).
